Kata-JS
=======

Kata in Javascript.

Install
-------
1. Install Gulp

   ```npm install -g gulp```
   
2. Run node install

    ```npm install```
    
3. (optionally) install tsd

    ```npm install tsd -g```

Run
---

To run tests:

```gulp test```

Useful commands
---------------
    tsd query <module> -a install --save
